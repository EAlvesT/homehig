# HOME HIGIENIZAÇÃO DE ESTOFADOS

Este projeto se trata de uma Landing Page para higienização de estofados na região de Volta Redonda e proximidades, nela, utilizei conhecimentos das seguintes tecnologias:

* HTML  
* CSS  
* JAVASCRIPT  
* TAILWIND  
* NEXT  
* REACT

Mantendo a construção de componentes de maneira reutilizada em alguns pontos do projeto, usando código limpo e de fácil compreensão com alguns recursos do Javascript ES6.

Neste projeto, alguns conteúdos não ficam diretamente no código fonte, foi utilizado um CMS chamado CosmicJs para gestão de conteúdo, onde o cliente define alguns serviços enviando imagem e texto.

Para acessar o site, [clique aqui](https://homehigvr.com.br).
